# My Paper Title

 DTI-MHAPR:Optimized Drug-Target Interaction Prediction via PCA-Enhanced Features and Heterogeneous Graph Attention Networks


## Requirements

To install requirements:

```
pip install -r requirements.txt
```

## Feature Extraction Training

To train the model as described in the paper, execute the following command:

```
python main.py
```

## Prediction

To obtain the final predictions using the trained model on the extracted features, run the following command:

```
python .\models\ML_model.py
```
